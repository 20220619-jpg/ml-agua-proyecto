Machine Learning aplicado a la calidad del agua

Clasificación y predicción mediante SVM y Random Forest

1. Introducción

Este proyecto aplica técnicas de Machine Learning para el análisis de la calidad del agua y la eficiencia de tratamiento de aguas residuales, utilizando modelos de Clasificación (SVM) y Regresión (Random Forest).

2. Clasificación de calidad de agua – SVM

El modelo SVM clasifica la calidad del agua a partir de parámetros fisicoquímicos como pH, turbidez y conductividad.

3. Predicción de eficiencia de tratamiento – Random Forest

El modelo Random Forest permite estimar la eficiencia de remoción (%) en plantas de tratamiento, considerando variables como DBO, DQO, SST y caudal.

4. Conclusiones

  • Los modelos de Machine Learning permiten analizar de forma eficiente la calidad del agua.
  • Las aplicaciones interactivas facilitan la comprensión de los resultados.
  • La integración web mejora la comunicación de los modelos desarrollados.
